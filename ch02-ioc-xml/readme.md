### 使用IOC，由Spring创建对象

di : 依赖注入，表示创建对象，给属性赋值

di的实现语法有两种
- 使用标签和属性完成，叫做基于XML的di实现
- 使用spring中的注解，完成属性的赋值，交叫做基于注解的di实现

di的语法分类：
- set注入（设置注入），80%
- 构造注入，调用有参构造方法完成赋值

实现步骤

- 创建Maven项目
- 加入Maven的依赖
    - Spring依赖
    - junit依赖
- 创建类（接口和它的实现类）
- 创建Spring需要使用的配置文件 声明类的信息，这些类由Spring创建和管理
- 测试Spring创建的对象
